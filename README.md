# fmod_wanderingmind

Download fmodapi from [drive](https://drive.google.com/file/d/1x6vUOR-Wd_HBsjHQo_IHatQkx6xeQoOR/view?usp=sharing) or from fmod [site](://fmod.com/download)

unzip in folder:
`tar -xf fmodstudioapi20201linux.tar.gz`

build project
`make build`

run with
`make run`

clean with
`make clean`
